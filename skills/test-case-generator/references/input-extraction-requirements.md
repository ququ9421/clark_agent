# 输入源提取 — 需求文档（Markdown / Word / 纯文本）

---

## 1. Markdown / PRD 文档提取规则

### 结构识别

| Markdown 元素 | 提取内容 |
|--------------|---------|
| `# / ##` 标题 | 功能域名称（→ TC ID 中的 `{FEATURE}`） |
| `### / ####` 子标题 | 功能点、用户故事 |
| 有序/无序列表 | 功能点枚举、验收标准 |
| 粗体文本 `**...**` | 关键术语、状态值、重要约束 |
| 表格 | 字段定义、枚举值、状态转换 |
| 代码块 ` ``` ` | API 路径、请求体示例、系统约束 |
| 引用块 `>` | 背景说明、补充约束 |

### 验收标准提取

识别以下关键词后的内容为验收标准（Acceptance Criteria）：

```
- "验收标准"、"AC-"、"Acceptance Criteria"
- "需满足"、"必须"、"应该"、"须"
- "Given/When/Then" BDD 格式
- "✅"、"✓" 标记的列表项
```

### criterionId 映射

当文档中有显式 AC 编号时，提取并写入 handoff 的 `criterionId` 字段：

```markdown
# 用户登录

## 验收标准
- AC-001: 有效邮箱和正确密码可以成功登录
- AC-002: 密码连续错误 5 次后锁定账号 30 分钟

→ 对应用例：criterionId: "AC-001" / "AC-002"
```

若无显式编号，`criterionId: null`。

## 2. 纯文本 / Word 需求转换规则

遇到以下格式时，**先转换为用户故事**，再进入标准流程：

| 原始格式 | 转换规则 | 示例 |
|---------|---------|------|
| `1. 系统应支持用户上传文件` | 作为用户，我希望上传文件，以便保存数据 | "应支持" → 用户视角 |
| `功能要求：密码须包含大小写字母` | 作为用户，我希望设置符合安全规则的密码，以便通过验证 | "须/需" → 目标/原因 |
| `必须支持批量操作` | 作为用户，我希望批量操作多条记录，以便提高效率 | "必须" → 动机/价值 |
| `管理员可以查看所有用户数据` | 作为管理员，我希望查看所有用户数据，以便管理账号 | 主语 → 角色 |

**规则**：无法明确识别的约束性语句，默认转换为正向用户故事。

## 3. 用户故事格式提取

标准格式：`作为 {角色}，我希望 {行为}，以便 {价值/原因}`

**提取映射**：
- `{角色}` → 前置条件中的用户类型（普通用户/管理员/游客）
- `{行为}` → 功能点名称（→ TC ID 的 `{FEATURE}` 部分）
- `{价值/原因}` → 测试目标（确认预期结果的业务语义）

---

## 4. Linear Issue 提取规则

从 Issue 描述中提取：

| Issue 字段 | 提取内容 |
|-----------|---------|
| 标题 | 功能点名称 |
| 描述 / 复现步骤 | 操作序列（→ 场景用例的步骤） |
| 验收标准字段 | 验收标准（→ criterionId + assertions） |
| Label / Priority | 映射到 P0/P1/P2 |
| Attachments（截图） | 补充 UI 元素识别 |

**优先级映射**：

| Linear Priority | Handoff priority |
|----------------|-----------------|
| Urgent | P0 |
| High | P1 |
| Medium | P1 |
| Low | P2 |
| No priority | P2 |

---

## 5. 功能域名称（{FEATURE}）提取规则

`{FEATURE}` 用于 TC ID 格式 `TC-{SOURCE}-{FEATURE}-{NNN}`，必须是大写英文。

**提取策略**（优先级从高到低）：

```
1. 用户显式指定（"这是登录功能的测试"）→ 直接使用
2. 文档一级标题（# 登录功能 → "LOGIN"）
3. 用户故事行为动词名词（"登录"→ LOGIN，"创建任务"→ TASK）
4. Linear Issue 类别 / Component 字段
5. 无法识别 → 询问用户
```

**常见映射**：

| 中文功能 | {FEATURE} |
|---------|-----------|
| 登录/认证 | LOGIN |
| 注册 | REGISTER |
| 任务/工作项 | TASK |
| 文件/附件 | FILE |
| 用户管理 | USER |
| 权限/角色 | RBAC |
| 搜索/筛选 | SEARCH |
| 通知 | NOTIFY |
| 支付/订单 | PAYMENT |
| 设置/配置 | SETTINGS |
